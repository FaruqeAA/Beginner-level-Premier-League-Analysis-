# Beginner-level-Premier-League-Analysis-
A simple analysis on the 23/34 premier league season using python from someone learning python from scratch.

#What exactly is this project
Its a summary of how each of the teams performed in the 23/24 season with a few charts . Covers points such as total goals, Home vs Away wins for every team across the league, Which teams drew the most, the best and worst defences and some other key season statistics.
#Findings 
-Man city scored the most goals establishing their title dominance 
-Arsenal had the best defence conceding less than 30 goals the entire season
- Brighton were the draw specialists
- 1246 total goals and the average of 3.28 per game

#Tools used 
- Python
- Pandas — data manipulation
- Matplotlib — data visualisation
- Google Colab — development environment
